Imagine-IoT! maker files repo!
=======================================

#### Imagine IOT Nodes firmware

## Atmega 328p Boards
* IMN00x Nodes
* Imagine Bootloader
* Boardfiles

## Nordic nRF52840 Boards
* RAK WisBlock LPWAN Module (RAK4631)
* TTN Payload decoder (WisBlock)

## ATTiny84 Boards
* ATTNode ATTiny84
* Case (3D)
* PCB Board files

* Alternating Flashing LEDs
* Rotating LEDs

* Random Flashing LEDs

### NUCLEO-L053R8
* Blinky

### NUCLEO-G071RB
* Blinky
